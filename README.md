# ことばのクエストギルド

朝の状況を自然な言葉で伝えると、ホコリス・モグモグ・ハコブンの3体が同時に出撃する Unity ミニゲームです。

## 起動

Unity 6000.4.5f1 で `unity/Assets/Scenes/Main.unity` を開き、Play を押してください。入力欄の依頼を変えて「ギルドに依頼する」を押すと、依頼を3体に分解して並列実行します。ネットワークなしでもデモできるよう、ゲーム内にはキーワードベースの安全なフォールバックがあります。

## OpenAI を使う生成プラン

リポジトリの **Actions → Generate Guild Plan → Run workflow** から依頼を入力すると、`OPENAI_API_KEY` シークレットで OpenAI Responses API に依頼を送り、3ロボット用の JSON プランを artifact として取得できます。

秘密鍵は Unity / WebGL ビルドには含めません。ブラウザゲームから OpenAI を直接呼ぶと鍵が公開されるためです。本番でリアルタイムのAI分解を行う場合は、このワークフローと同じ環境変数の扱いで、認証付きのサーバー側 API を追加してください。
